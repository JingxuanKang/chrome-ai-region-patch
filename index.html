Chrome AI Region Patch

A tiny local script for restoring Chrome's built-in AI eligibility fields after browser updates rewrite regional state.

What It Does

Chrome can reset local region fields during updates. This tool patches Local State so built-in AI surfaces such as Gemini in Chrome, AI history search, and DevTools AI can become eligible again.

python3 chrome_ai_region_patch.py --force-kill

Patch Model

Local only

Reads and writes Chrome configuration files on disk. No network calls, extensions, cookies, or account data.

Version aware

Uses the installed Chrome app version first, avoiding stale profile version values after updates.

Recoverable

Creates a timestamped backup of Local State before writing changes.

Fields

The script sets is_glic_eligible to true, writes variations_country as us, and updates variations_permanent_consistency_country with the current Chrome version and region.
